HTML Tags

Below is a list of five HTML tags learned in class:

html
Indicates the beginning and end of an HTML element in an HTML document.
article
It is used for containing independent piece of content. It could be shared as stand alone piece of content and will still make sense.
p
It is used to add a paragraph of text. The browser will add a space before and after the content of the paragraph.
br
It creates a significant line break to the content. Useful in poems and addresses where the division of lines is important.
div
It is used to wrap around multiple child elements. Div divides the content into logical groups which will show the content on a separate line.
